Anonymous Censorship-Resistant Messaging

secIRC is a completely anonymous messaging system that bypasses any kind of censorship through its distributed relay network. Messages are end-to-end encrypted and relay servers don't store metadata about origin or destination.

How It Works

1

Generate Identity

Create a cryptographic keypair and generate a unique hash identifier. No personal information required.

2

Discover Relays

Use BitTorrent-inspired DHT and trackers to find relay servers in the distributed network.

3

Send Messages

Encrypt messages with recipient's public key and route through random relay chains.

4

Receive & Decrypt

Only the recipient can decrypt messages using their private key. Relays never see the content.

async def send_anonymous_message(
    recipient_hash: bytes,
    content: bytes,
    sender_private_key: bytes
) -> bool:
    # Encrypt with recipient's public key
    encrypted_content = encrypt_message(
        content, 
        get_public_key(recipient_hash)
    )
    
    # Route through random relay chain
    relay_chain = select_random_relays(3)
    return await route_message(
        encrypted_content,
